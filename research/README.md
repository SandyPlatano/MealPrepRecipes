# Research Archives

UX and product research conducted for MealPrepRecipes.

## Directories

| Directory | Focus |
|-----------|-------|
| `ux-ui/` | General user experience research |
| `settings/` | Settings architecture research |
| `social/` | Social recipe sharing research |
| `household/` | Household collaboration patterns |
| `shopping/` | Shopping list intelligence |
| `mobile-first/` | Mobile-first cooking and shopping UX |
| `mobile-cooking/` | Mobile cooking shopping UX research |
| `meal-planning-competitive/` | Competitive landscape analysis |

## Structure

Each research directory typically contains:
- `objective.md` — Research goals and questions
- `evidence/` — Raw research data and sources
- `persona-findings/` — User persona analysis
- `synthesis/` — Consolidated findings
- `report.md` — Final research report

## Usage

This research informs feature development and UX decisions. Reference these findings when:
- Planning new features
- Evaluating user flows
- Making design decisions
- Understanding user needs
