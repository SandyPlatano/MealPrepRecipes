## Contributing

When adding documentation:
1. **Active docs** go in the appropriate `/docs/` subdirectory
2. **Feature requirements** go in `/.docs/plans/[feature-name]/requirements.md`
3. **Completed/historical docs** go in `/docs/archive/`
4. **Research** goes in `/research/`
