## Getting Started

```bash
cd nextjs && npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser.

## Tech Stack

- **Framework:** Next.js 16.1 (App Router)
- **Database:** Supabase
- **Styling:** Tailwind CSS + shadcn/ui
- **AI:** Anthropic Claude
